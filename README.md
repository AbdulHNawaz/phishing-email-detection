# Phishing Email Detection

This project compares machine learning and deep learning models for phishing email detection.

## Technologies
- Python
- PyTorch
- Scikit-learn
- Pandas
- NumPy

## Models
- Logistic Regression
- Support Vector Machine (SVM)
- Naive Bayes
- CNN
- LSTM
- DistilBERT

## Results
- SVM achieved 97.2% accuracy.
- CNN achieved 99.0% recall.
- DistilBERT was evaluated and its performance analysed against the other models.

## Dataset
Public phishing email dataset from Hugging Face.

## Skills Demonstrated
- Data preprocessing
- Feature engineering
- Model training
- Hyperparameter tuning
- Model evaluation using Precision, Recall, F1-score, and Confusion Matrix
